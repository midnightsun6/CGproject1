Shader Path:
	預設路徑：./res/shaders/
	填寫例：shader.fs

Object Path:
	預設路徑：./res/models/
	填寫例：Android Robot/Android Robot.obj
	
Animation Path:
	預設路徑：./res/animations/
	填寫例：test.objani
	
Animation File format(.objani):
[Animation Name](string)
[Duration Time](float)
[Is Loop](bool)
[Play Speed](float)
[A number of Body Parts](int)
[Body Part1](string)
[A number of Keyframes](int)
[Keyframe1, posX, posY, posZ, rotX, rotY, rotZ, scaX, scaY, scaY](float)
[Keyframe2, posX, posY, posZ, rotX, rotY, rotZ, scaX, scaY, scaY](float)
[A number of Keyframes](int)
[Body Part2](string)
[Keyframe1, posX, posY, posZ, rotX, rotY, rotZ, scaX, scaY, scaY](float)
[Keyframe2, posX, posY, posZ, rotX, rotY, rotZ, scaX, scaY, scaY](float)
.
.
.
